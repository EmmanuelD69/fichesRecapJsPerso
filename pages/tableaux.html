  • LES TABLEAUX

    Un tableau est un contenant qui possède une ou plusieurs valeurs aussi appelées "items".
    Chaque item est accessible au moyen d'un "indice" (index en anglais) qui commence à partir de 0.

    Le contenu du tableau se définit entre crochets "[...]" et chaque "item" est séparée par une virgule.

    Pour schématiser:
    Un tableau c'est comme un catalogue dans lequel on trouve à telle page tel produit.

    En version Javascript cela donne:
    Un tableau c'est un contenant où l'on trouve à tel indice telle valeur.

    Ces valeurs peuvent être: un nombre, une chaine de caractères, un objet, une fonction, ...

    Examples:
    const listStrings = ['Sébastien', 'Laurence', 'Ludovic','Pauline', 'Guillaume'];
    indice 0 = Sébastien
    indice 1 = Laurence
    indice 2 = Ludovic
    indice 3 = Pauline
    indice 4 = Guillaume

    const listNumbers = [5, 10, 15, 20, 25];
    indice 0 = 5
    indice 1 = 10
    indice 2 = 15
    indice 3 = 20
    indice 4 = 25

    const listObjets = [user1, user2, user3, user4, user5];
    indice 0 = user1
    indice 1 = user2
    indice 2 = user3
    indice 3 = user4
    indice 4 = user5

    const listFonctions = [max(1, 5), min(1, 5)];
    indice 0 = max(1, 5)
    indice 1 = min(1, 5)

    const list = [10, 'Laurent', user, max(20,50)];
    indice 0 = 10
    indice 1 = Laurent
    indice 2 = user
    indice 3 = max(20, 50)

    Pour cibler un item, il suffit de nommer le tableau auquel il appartient
    et de mettre à la suite, entre crochets, l'indice correspondant à la position de l'item dans le tableau.

    Soit un tableau d'utilisateurs:

    const listUsers = ['Sébastien', 'Laurence', 'Ludovic', 'Pauline', 'Guillaume'];
    console.log(listUsers[2]);

    /* nous affichera Ludovic dans la console. */

    "length" est une propriété qui permet d'afficher le nombre d'éléments que comporte un tableau.

    Example:

    const listUsers = ['Sébastien', 'Laurence', 'Ludovic', 'Pauline', 'Guillaume'];
    console.log(listUsers.length);

    /* nous affichera 5 dans la console. */

  • METHODES FREQUEMMENT UTILISEES AVEC LES TABLEAUX

    les méthodes sont des fonctions, et peuvent donc recevoir des paramètres.

    Il est possible d'interagir avec un tableau de plusieurs façons,
    pour cela on utilise, comme pour intéragir avec un objet, un point (.)
    suivi de la méthode que l'on souhaite appliquer à notre tableau.

    "push()" permet d'ajouter un ou plusieurs items à la fin d'un tableau,
    "push()" peut recevoir un nombre illimité de paramètres,
    et chaque paramètre représente un item à ajouter à la fin du tableau.

    const myFriends = ['Sébastien', 'Laurence'];

    myFriends.push('Ludovic');
    /* Ajoute « Ludovic » à la fin du tableau */

    myFriends.push('Pauline', 'Guillaume');
    Ajoute « Pauline » et « Guillaume » à la fin du tableau

    "unshift()" fonctionne comme "push()",
    excepté que les items sont ajoutés au début du tableau.
    Cette méthode n'est pas très fréquente mais peut être utile.

    const myFriends = ['Sébastien', 'Laurence'];

    myFriends.unshift('Ludovic');
    /* Ajoute « Ludovic » au début du tableau */

    myFriends.unshift('Pauline', 'Guillaume');
    Ajoute « Pauline » et « Guillaume » au début du tableau

    "shift()" et "pop()" retirent respectivement le premier et le dernier élément du tableau.

    Soit le tableau:
    const listUsers = ['Sébastien', 'Laurence', 'Ludovic', 'Pauline', 'Guillaume'];

    listUsers.shift(); // Retire le premier indice et sa valeur: « Sébastien »
    listUsers.pop(); // Retire le dernier indice et sa valeur: « Guillaume »
    alert(listUsers); // Affichera « Laurence, Ludovic, Pauline »

    "split()": appliqué à une variable contenant une chaine de caractères,
    cette méthode permet de découper cette string en un tableau de valeurs.

    Le découpage se fait en fonction du paramètre que l'on passe (ou pas) à la méthode split.
    Dans notre example, on passe en paramètre un "espace",
    ce qui indique à la méthode qu'il lui faut prendre en compte chaque espace
    qu'elle va rencontrer pour construire le tableau.
    On est donc passé d'une variable contenant une chaine de caractères
    à une variable contenant un tableau d'items.

    Soit une variable "myFamily" contenant une string:

    let myFamily = 'Pauline Guillaume Clarisse';
    console.log(myFamily);
    /* affichage en console: Pauline Guillaume Clarisse */

    myFamily = myFamily.split(' ');
    console.log(myFamily);
    /* affichage en console: ["Pauline", "Guillaume", "Clarisse"] */

    "join()": appliqué à une variable contenant un tableau d'items,
    cette méthode va transformer le contenu de la variable en une chaine de charactères.

    Même principe pour le découpage que pour la méthode split() mais en sens inverse.
    Dans notre cas, un espace séparera chaque items rencontrés dans le tableau pour en faire une chaine de caractères.

    Soit une variable "myFamily" contenant un tableau:

    let myFamily = ["Pauline", "Guillaume", "Clarisse"];
    console.log(myFamily);
    /* affichage en console: ["Pauline", "Guillaume", "Clarisse"] */

    myFamily = myFamily.join(' ');
    console.log(myFamily);
    /* affichage en console: Pauline Guillaume Clarisse */

    "indexOf()" permet d'afficher l'indice de l'item que l'on a passé en paramètre.

    Example:

    const listUsers = ['Sébastien', 'Laurence', 'Ludovic', 'Pauline', 'Guillaume'];
    console.log(listUsers.indexOf('Laurence'));

    /* nous affichera 1 dans la console. */

  • HIGHER ORDER FUNCTION / FONCTION DE HAUT RANG

    Qu'est-ce que c'est??
    C'est une fonction qui utilise une autre fonction en paramètre/argument.

    Example:
    Const mixes = ["Techno", "House", "Disco", "Funk"];

    mixes.forEach(function(mix)){
     console.log(mix);
    }

    la fonction "forEach" utilise une fonction anonyme en paramètre/argument.
    Une fonction anonyme est une fonction qui n'a pas de nom.

    Autre example:

    on crée une fonction, de type "High order", que l'on nommera "repeater".
    Elle prendra en paramètre une autre fonction.

    1-Création de la fonction "repeater":
    funtion repeater(function) {
     function();
     function();
     function();
    }

    2-Création de la fonction "sayHello":
    function sayHello(){
     console.log("Hello There!");
    }

    3-Appel de la fonction "repeater" en lui passant la fonction "sayHello" en argument:
    repeater(sayHello);

    Le résultat de la fonction repeater sera d'afficher
    trois fois "Hello There!" dans la console.

  • CALLBACK FUNCTION / FONCTION DE RAPPEL

    Qu'est-ce que c'est??
    C'est une fonction qui est utilisée à un moment précis,
    déterminé par une autre fonction (high order function)
    ou par une action précise (addEventListener).

    Pour résumer c'est la fonction que l'on passe en argurment/paramètre d'une autre fonction(higer rank function).

    Example1:

    Const mixes = ["Techno", "House", "Disco", "Funk"];

    mixes.forEach(function(mix)){
     console.log(mix);
    }

    function (mix) est notre fonction callback

    Example2:

    button.addEventListener("click", function(){
     console.log("Hello There!");
    });

    function() est notre fonction callback

  • "MAP" Qu'est-ce que c'est??

    Une méthode qui appliquée à une variable contenant un tableau,
    permet de cloner le contenu de la variable et d'y apporter d'éventuelles modifications
    sans pour autant modifier le tableau d'origine.

    Son fonctionnement est similaire à FOR EACH, On boucle sur un tableau.

    Soit une variable "mixes" contenant un tableau:
    const mixes = ["Techno", "House", "Disco", "Funk"];

    1- Création de la variable newMixes qui va cloner grâce à la méthode
    "map()" le tableau contenu dans la variable mixes.
    On décide d'appliquer à ce nouveau tableau, par l'intermédiaire d'une fonction anonyme (callback),
    une modification des items en les passant en Majuscule (toUpperCase).

    const newMixes = mixes.map(function(mix){
     return mix.toUpperCase();
    });

    2- Affichage du tableau contenu dans la variable d'origine:
    console.log(mixes);
    Array(4) [ "Techno", "House", "Disco", "Funk" ]

    3- Affichage du tableau contenu dans la nouvelle variable:
    console.log(newMixes);
    Array(4) [ "TECHNO", "HOUSE", "DISCO", "FUNK" ]

    4- On "push"/ajoute un nouvel item dans le tableau de la nouvelle variable(newMixes):
    newMixes.push("Deep");

    5- Affichage du tableau contenu dans la variable newMixes:
    console.log(newMixes);
    Array(5) [ "TECHNO", "HOUSE", "DISCO", "FUNK", "Deep" ]

    6- Affichage du tableau contenu dans la variable d'origine:
    console.log(mixes);
    Array(4) [ "Techno", "House", "Disco", "Funk" ]

    On constate que le tableau d'origine n'a pas changé tandis que le nouveau à bien été modifié.

    Conclusion:
    MAP NOUS PERMET DE CLONER UNE VARIABLE CONTENANT UN TABLEAU ET
    DE MODIFIER LE CONTENU DE LA NOUVELLE VARIABLE
    SANS POUR AUTANT CHANGER CELUI DE LA VARIABLE D'ORIGINE.

  • "FIND" Qu'est-ce que c'est??

    Une méthode qui parcourt un tableau et compare chaque élément(item) à une condition.
    Si la condition est respectée elle nous retourne une valeur, si ce n'est pas le cas elle nous retourne "undefined".

    Example:
    Soit une variable mixes contenant un tableau:
    const mixes = ["Techno", "House", "Disco", "Funk"];

    const searchMix = mixes.find(function (mix) {
    return mix.includes("Disco");
    });

    On applique la méthode find() à la variable mixes dont la fonction anonyme(callback)
    à pour rôle de renvoyer la valeur du premier élément/item qui correspondra à la condition qu'on a fixée.
    Dans le cas présent, le premier item qui inclut la chaine de charactères "Disco".

    console.log(searchMix);

    La console va bien nous afficher le mot "Disco" car il existe bien dans notre tableau.
    S'il n'avait pas existé, find() nous aurait renvoyé "undefined".

    FIND RENVOI UNIQUEMENT LE PREMIER ITEM CORRESPONDANT A
    LA CONDITION, PUIS SORT DU TABLEAU.

  • "FILTER" Qu'est-ce que c'est??

    C'est une méthode qui fonctionne exactement comme find() à la différence que
    filter() parcourt un tableau dans son intégralité
    et ne s'arrête pas à la première valeur trouvée correspondant à la condition fixée.

    Example:
    Soit une variable mixes contenant un tableau:
    const mixes = ["Techno", "House Music", "Disco House", "Funk"];

    const searchMix = mixes.filter(function (mix) {
    return mix.length > 9;
    });

    Une seule condition à la fois, après un return on ne peux pas enchainer de code dans le scope de la fonction

    On applique la méthode filter() à la variable mixes dont la fonction anonyme(callback)
    à pour rôle de renvoyer la valeur de chaque item qui correspondra à la condition qu'on lui a fixée.
    Dans le cas présent, tous les items dont la longueur est supérieur à 9 caractères.

    console.log(searchMix);

    La console va nous afficher les items "House Music" et "Disco House" car ils contiennent bien la string "House".
    S'il n'avait pas existé d'item contenant la string "House" nous aurions eu en retour une valeur "undefined".

    FILTER PARCOURT L'ENSEMBLE D'UN TABLEAU ET AFFICHE
    TOUS LES ELEMENTS/ITEMS CORRESPONDANT A LA CONDITION.

  • "SOME" ET "EVERY" Qu'est-ce que c'est??

    Méthodes qui permettent de parcourir et comparer chaque éléments d'un tableau à une condition.
    Selon que la condition soit respecté ou non, on aura un retour "True" ou "False".
    Le but de "some()" et "every()" est de renvoyer "true" ou "false".
    SOME renverra "true" si certains des items du tableau sont vérifiés comme "true" par la condition.
    EVERY renverra "true" uniquement si tous les items du tableau sont vérifiés comme "true" par la condition.

    Soit "games" un tableau d'objets:

    Rappel: Un objet est un contenant possédant une ou plusieurs propriétés
    une propriété est l'association d'un nom(key) et d'une valeur(value).

    const games=[
     { title: 'Mass Effect', rating: 9.5 },
     { title: 'League of Legends', rating: 5 },
     { title: 'Last of Us', rating: 10 },
     { title: 'God of War', rating: 10 },
     { title: 'WWE 2k20', rating: 4 },
    ];

    Example some():
    const checkRating = games.some(function (game) {
     return game.rating > 8;
    });

    Une seule condition à la fois, après un return on ne peux pas enchainer de code dans le scope de la fonction

    On applique la méthode some() à la variable games dont la fonction anonyme(callback)
    à pour rôle de vérifier une condition et de nous retourner soit "true" ou "false".
    Dans le cas présent, on verifie s'il existe des propriétés dont la valeur associé à la key "rating" est supérieur à 8.

    console.log(checkRating);

    La console va nous afficher "true" car quelques proriétés ont bien une valeur rating supérieur à 8.
    Si nous n'avions pas trouvé de valeurs rating supérieur à 8 sur l'ensemble des objets du tableau, nous aurions eu "false".

    Example every():
    const checkRating = games.every(function (game) {
     return game.rating > 8;
    });

    Une seule condition à la fois, après un return on ne peux pas enchainer de code dans le scope de la fonction

    On applique la méthode every() à la variable games dont la fonction anonyme(callback)
    à pour rôle de vérifier une condition et de nous retourner soit "true" ou "false".
    Dans le cas présent, on verifie si toutes(every) les propriétés dont la valeur associé à la key "rating" sont supérieur à 8.

    console.log(checkRating);

    La console va nous afficher "false" car quelques propriétés ont des valeurs rating inférieur à 8.
    Si toutes les valeurs avait été supérieur à 8, nous aurions eu "true".
